# Website Hợp tác xã Nông nghiệp Dịch vụ Công bằng EaTu

Đây là một dự án website tĩnh được xây dựng để giới thiệu về Hợp tác xã Nông nghiệp Dịch vụ Công bằng EaTu, chuyên cung cấp các sản phẩm cà phê chất lượng cao tại Đắk Lắk, Việt Nam.

## Demo trực tiếp

Bạn có thể xem trang web hoạt động tại đây: [https://vinhphannn.github.io/htxeatu/](https://vinhphannn.github.io/htxeatu/)

## Tính năng chính

- **Giao diện Responsive:** Tương thích tốt trên các thiết bị từ mobile, tablet đến desktop.
- **Trang Giới thiệu (Về HTX):** Cung cấp thông tin chi tiết về lịch sử, sứ mệnh, và các chứng nhận của HTX (FLO, VietGAP, OCOP).
- **Trang Sản phẩm:**
  - Hiển thị danh sách sản phẩm cà phê đa dạng.
  - Phân loại sản phẩm theo danh mục (pha phin, pha máy, hòa tan, cho quán).
  - Chức năng tìm kiếm sản phẩm.
- **Trang Liên hệ:** Tích hợp form liên hệ và thông tin chi tiết của HTX.
- **Hiệu ứng động:** Sử dụng thư viện AOS (Animate on Scroll) để tạo hiệu ứng mượt mà khi người dùng cuộn trang.
- **Chatbox hỗ trợ:** Một chatbot đơn giản để trả lời các câu hỏi thường gặp của khách hàng.
- **Tích hợp Google Maps:** Hiển thị vị trí của HTX trên bản đồ.

## Bắt đầu

Đây là một dự án web tĩnh, bạn không cần cài đặt phức tạp. Chỉ cần làm theo các bước sau:

1. **Clone repository về máy của bạn:**
   ```bash
   git clone https://github.com/vinhphannn/htxeatu.git
   ```
2. **Mở file `index.html`:**
   - Điều hướng đến thư mục dự án vừa clone.
   - Mở file `index.html` bằng trình duyệt web yêu thích của bạn.
